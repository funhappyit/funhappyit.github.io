## 오늘 한 일

### 그리드 라이브러리 이슈 대응
- 서버 페이징 + 마스터-디테일이 함께 켜진 DevExtreme 그리드에서 엑셀 "전체 다운로드"가 조용히 멈추는 문제를 진단하고 수정
  - 원인 ①: 하위 상세 행이 펼쳐진 상태로 `dataSource`를 교체하면서 발생하는 크래시
  - 원인 ②: `exportDataGrid()` + `dataSource` 스왑 조합이 서버 페이징·마스터-디테일 조합에서 응답 없이 멈추는 문제
  - 해결: 다운로드 전 펼쳐진 행 자동 접기 + 요약 시트도 ExcelJS로 직접 기입하는 방식으로 전환 + `try/catch` 안전망 추가
  - 자세한 내용은 [바로 이전 포스트]({{ site.baseurl }}{% post_url 2026-07-10-devextreme-export-hang-fix %}) 참고
- 수정 사항을 PR로 올리고 셀프 코드리뷰 진행 — 아래 개선점 발견
  - 동작 변경(항상 상세 시트 포함) 사항이 PR 설명에 누락되어 있었음
  - 요약 시트 컬럼 너비가 고정값으로 바뀌면서 이전의 콘텐츠 기반 자동 너비 조정 기능이 빠짐
  - 예외 처리(`try/catch`)가 일부 데이터 조회 단계까지는 커버하지 못함

### 사이드 프로젝트
- GitHub Pages 기반 개인 개발 블로그(Jekyll) 신규 구축
  - "오류 해결" / "개발 작업" 두 카테고리로 글을 정리하는 구조 설계 (`_config.yml`, `index.md`에 카테고리별 목록 반영)
  - 오늘 다룬 오류 해결 내용을 첫 포스트로 정리해 게시
  - 나중에 재사용할 수 있도록, 대화 맥락에서 오류/작업 내용을 뽑아 블로그 포스트 형식으로 정리해주는 스킬(SKILL) 초안 작성
